🎯 KBC-Style Quiz Game in Python

This Python program simulates a Kaun Banega Crorepati (KBC)-style quiz game,
where players answer multiple-choice questions to win increasing amounts of virtual prize money.

It’s a great example of using lists, loops, and conditional logic in Python to create an interactive console-based game.

🧠 Features

16 multiple-choice questions stored in lists

Dynamic question and option display using loops

Correct answer validation

Incremental prize system

Game automatically ends on the first wrong answer

Displays total prize won

🧩 Example Gameplay
01. What is the color of the sky on a clear day?
A) Green
B) Blue
C) Purple
D) Red

Enter the option (A/B/C/D)---> B
✅ Correct!

02. Which animal is known as the “king of the jungle”?
A) Elephant
B) Tiger
C) Lion
D) Cheetah

Enter the option (A/B/C/D)---> C
✅ Correct!

Congratulations You Won ₹3000

💡 Concepts Used

List data type for questions, options, correct answers, and prize money

for loops for iteration

if-else conditions for validation

input() for user responses

Basic arithmetic and logical control

🛠 How to Run
python kbc_quiz_game.py
